drawstyle Built-in Style Seed Runbook

Use this after production deploy and OIDC admin login are working. Cloudflare commands must use wrangler-accounts, not bare wrangler.

Prerequisites

CheckExpected
Admin loginADMIN_EMAILS contains the email returned by account.leeguoo.com.
DatabaseD1 binding points to shared public-db; tables are prefixed drawstyle_.
AssetsR2 bucket drawstyle-assets exists.
CLI repo/Users/leo/github.com/chatgpt-imagegen has built-in snippets and example images.

Steps

  1. Open https://drawstyle.leeguoo.com/submit and log in as an admin account.
  2. Submit each built-in style with slug/name matching the style key.
  3. Paste the snippet from the CLI source, attach 1-3 example images, and attach any reference images that should ship to CLI pulls.
  4. Open /admin, approve each submission, and verify it appears on the gallery.
  5. Smoke-test one pull from the real CLI:
DRAWSTYLE_API=https://drawstyle.leeguoo.com chatgpt-imagegen style pull doodle --as doodle2
